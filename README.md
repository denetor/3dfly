# three.js demo
Demo che dovrebbe mostrare un terreno, dei palazzi di diverse dimensioni e la telecamera in grado di spostarsi
nel mondo 3d

## Startup
Avviare il sistema con:
```bash
npm run dev
```

Vedere il progetto navigando in: `http://localhost:5173`.
